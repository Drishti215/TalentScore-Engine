# TalentScore Engine

An ML-powered Streamlit app that analyzes resumes against job descriptions to highlight:
-  ATS Match
-  Skill Gaps
-  Recruiter Attention
-  Keyword Insights
-  AI Suggestions

##  Features
- Upload resume and job description CSVs
- Get ATS match percentage
- Identify missing and extra skills
- Visual recruiter attention map
- Keyword density analysis
- AI-powered improvement suggestions

##  Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
- Plotly

##  Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
